Place member photos in this folder.

Photo naming convention:
- Use the filename specified in the member's MD file
- Recommended size: 300x300 pixels or larger
- Supported formats: .jpg, .png, .webp
- Example: student1.jpg, professor.png

Default avatar:
- If a member doesn't have a photo, use "default-avatar.jpg"
- You can replace this with your own default avatar image
